Grunt-boilerplate

Very simple Grunt boilerplate for web apps or website develpment

View project onGitHub

INSTALLATION : Navigate to your project directory and type $ git clone https://github.com/benjaminhorner/grunt-boilerplate

Launch with : $ grunt Build with $ grunt build
